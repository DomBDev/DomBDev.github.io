# DomBDev.github.io

Personal portfolio site for Dominic Bonanni.

Built to be a clean public facing hub for my background, projects, and contact info, with a stronger focus on backend heavy work even though the site itself is static to ensure responsiveness and clean behavior.

## Structure

* `index.html` - single page portfolio with tabbed sections for About, Resume, Portfolio, and Contact
* `assets/css/` - page and section styling
* `assets/js/` - tab switching, interactive profile canvas, and small UI behavior
* `favicon/` - favicon and app icon files

## Local development

Serve the repo with any simple static server.

## Deployment

This site is deployed through GitHub Pages from this repo.

Push to `main` to update the live site.
